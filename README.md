## :speech_balloon: Sobre

O Spacetime é uma aplicação de recordação de memórias, onde é possível adicionar textos, fotos e vídeos em uma time line de acontecimentos que marcaram sua trajetória.

Esta aplicação foi desenvolvida durante o programa NLW, da [Rocketseat](https://www.rocketseat.com.br/ignite), composta por back-end, front-end e mobile.
